Convert hyper-v VM to VirtualBox VM

Date Fri 09 February 2024 Tags Tech / VM

Teaching on Youtube Video

step1: Export Hyper-V VM disk

Export the harddisk image from the Checkpoint that you prefer of Hyper-V Virtual Machine.

step2: Convertion

C:\Program Files\Oracle\VirtualBox> VBoxManage.exe clonemedium disk "src.vhdx" "des.vdi"

step3: Create VirtualBox VM based on "des.vdi"

  1. Settings => System => Acceleration => Hyper-V
  2. Settings => System => Motherboard => Enable EFI (checked)
  3. Settings => Display => Enable 3D Acceleration (checked)
