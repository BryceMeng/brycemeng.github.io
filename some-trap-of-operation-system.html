一些系统上的坑

Date Tue 06 August 2019 Tags Trap / System / Ubuntu

Ubuntu 更新显卡驱动后，鼠标键盘失灵

sudo apt install xserver-xorg-input-all

记住，现在ubuntu 18.04 安装完后默认是没有SSH Server的，一定要先把这个装好，不然远程操作不了电脑，只能用recover模式，会麻烦一些

如果没特征需求就用windows吧，可控性还是要高很多，而且win10也有WSL，ubuntu的适配性太差了，我有时想买新的笔记本都不敢买，万一ubuntu没有对应设备的驱动，启不......

一定要买ubuntu本的请看官方ubuntu笔记本检测列表

Ubuntu 18.04 经常桌面重启

表现为 xorg 进程 cpu 占单核100%，有时无响应，有时自动重启

我都快丧失信心了，经常重启会导致有些任务没有保存就退出了

某天搜索了一下，发现一个方法，然后就再也没重启过了，生活也美好了一些

安装 gnome-tweaks，并打开

点击菜单柆栏"优化"，再点击"重置为默认值"

重启电脑

整个世界都舒服了，所以我现在除了改桌面背景，其他任何设置都不敢更改了
